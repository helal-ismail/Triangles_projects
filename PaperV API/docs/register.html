Registration Request

URL

  1. http://paperv.com/api/register.php

Request Type

  1. GET/POST

Request Params

  1. user_name : username
  2. full_name : user full name
  3. email : user email
  4. password : user password

Response Format

  1. Success : {"success":true, "user_id":123, "msg":"User registered succesfully"}
  2. Fail : {"success":false, "msg":"Username or email already exists" }
